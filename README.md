
 🛠️ Projeto: Blackline Tattoo Studio — Landing Page Institucional

 🎯 O que é o projeto?

Este projeto é uma landing page institucional e autoral desenvolvida para o Blackline Tattoo Studio, um estúdio de tatuagem fictício especializado em artes exclusivas e personalizadas no Rio de Janeiro[cite: 1]. 

O objetivo principal da aplicação é funcionar como uma vitrine digital de alta conversão. O site resolve um problema real de negócio: centraliza as informações essenciais do estúdio, apresenta o portfólio e o estilo de cada artista (como Camila Varga, Rafael Nogueira e Marcelo Duarte) e direciona o usuário de forma fluida para o fechamento de um agendamento através de chamadas para ação (CTAs) estratégicas[cite: 1].

---

 🚀 Decisões Técnicas & Arquitetura

Para garantir a melhor performance e indexação possíveis, o projeto foi construído do zero utilizando tecnologias nativas (Vanilla Web), sem a sobrecarga de frameworks pesados:

*   **HTML5 Semântico:** Toda a estrutura da página foi desenhada pensando em SEO (otimização para motores de busca) e acessibilidade. Tags como `<header>`, `<section>`, `<main>` e `<footer>` foram utilizadas corretamente para garantir que o Google entenda perfeitamente a hierarquia do conteúdo[cite: 1].
*   **CSS3 Moderno & Responsivo:** O layout foi construído com foco em **Mobile-First**, utilizando Flexbox e CSS Grid para criar uma interface que se adapta perfeitamente desde telas de smartphones antigos até monitores desktop. A identidade visual adota uma estética escura (*Dark Mode* moderno), conversando diretamente com o público-alvo do estúdio de tatuagem[cite: 1].
*   **Foco em Performance (Core Web Vitals):** Ao optar por HTML e CSS puros, o site ganhou um tempo de carregamento extremamente baixo. As imagens dos artistas e dos trabalhos foram otimizadas para não comprometer a velocidade de renderização da página[cite: 1].

---

 💻 Funcionalidades Principais do Site

*   Seção Hero de Alto Impacto: Apresentação clara do propósito do estúdio logo no primeiro frame[cite: 1].
*   Galeria de Artistas Integrada: Espaço modular para exibir a foto, o nome e a especialidade técnica de cada tatuador do time[cite: 1].
*   Prova Social: Seção dedicada a depoimentos reais de clientes para gerar autoridade e confiança[cite: 1].
*   Funil de Agendamento: Botões de conversão (CTAs) estrategicamente posicionados ao longo da página, facilitando o contato direto via WhatsApp ou e-mail com a equipe de atendimento[cite: 1].
*   Rodapé Informativo: Centralização de canais de contato, endereço físico no Rio de Janeiro e links para redes sociais[cite: 1].
